# MVM Mission Generator

This is an early alpha replacement for mvm.tf.

The goal of this project is to make Mann vs. Machine mission editing and generation easier to use, especially for people who find the current mvm.tf interface difficult to navigate. The new site is still in active development, so expect rough edges, incomplete features, and layout changes as the project grows.

Any contributions are extremely helpful, whether that means bug fixes, UI improvements, data cleanup, sample mission tests, or ideas for making the workflow more usable.

## Current status

- Early alpha
- Work in progress
- Behavior and design may change without notice

## Getting involved

If you want to help, please open an issue or submit a pull request with whatever you can improve. Small or large contributions are welcome.
