# Contributing to MVM Mission Generator

Thanks for helping improve this project.

Small contributions are especially valuable. You do not need to make a huge change to be useful.

## What helps most

- Bug fixes
- UI and usability improvements
- Data cleanup and corrections
- Sample mission checks
- Small feature additions that keep the current architecture simple

## Project approach

- Keep the DOM as a view of JavaScript state, not the source of truth
- Prefer small, focused changes over large rewrites
- Preserve existing behavior unless a change is clearly an improvement
- Keep the code easy to read and easy to extend

## Before you submit

- Test the change locally
- Check that the UI still loads and works as expected
- If you add or change data, make sure the resulting object shape stays clean and explicit

## Good first contributions

- Fix text, labels, or layout issues
- Improve the bot editor workflow
- Add validation for bad input
- Expand sample data or examples
- Add features you enjoyed from mvm.tf

## Questions and ideas

If you are not sure whether something fits the project, open an issue. Helpful suggestions are welcome even if the implementation is small.
